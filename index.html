Image Hover Affects

CSS has a hover selector, :hover, which allows you to change certain elements upon the user hovering over that element. In the case of using hover to change images, you can use hover to simply change style the images differently, add image filters, animations, transitions, overlay effects, or a mix of all of it! Image hover affects can be used for a large variety of purposes, such as text popping up as a caption, description, and so on. In this case, I will detail the things you can do to edit an image while utilizing hover.

CSS has a property called filter:, which allows you to add various effects to images such as blur(px), brightness(%), contrast(%), dropshadow, grayscale(%), hue-rotate(deg), invert(%), opacity(%), saturate(%), and sepia(%). Other properties that can be used to change images are things such as border, border-radius, width, and height. Using transform, you can rotate the image or even flip it.

Take this picture of a building with brutalist architecture, using grayscale, transform, and trasition, I made it so it turned grey and slowly flips itself upside down upon hovering.
